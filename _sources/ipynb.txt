IPython Notebook
================

To get ipython notebook files working:

* `Install conda <http://conda.pydata.org/miniconda.html>`_ and run ``conda update conda``.

* Run these three commands:

.. code-block:: bash

   conda create -n ipy ipython ipython-notebook ipython-qtconsole

   source activate ipy
   
   ipython notebook


* Navigate to the local .ipynb file and open it.
